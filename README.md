# Alexander Wang — Portfolio 2.0

Single-file portfolio website. Dark/light theme, zh/en i18n, 6-page SPA, video gallery, 3D folder cards, AI assistant.

## Quick Start
Open `site.html` in a browser. Press `D` for dark mode.

## Project Structure
```
site.html          — Main website (self-contained HTML/CSS/JS)
task-board.md      — 13-step implementation roadmap
bug-tracker.md     — Bug tracking
design-docs/       — Design specs per task
fonts/             — Local font files
iQIYI-section.html — iQIYI case study module
金开心数据/         — NetEase analytics data
```

## Tech
- Vanilla HTML/CSS/JS (no frameworks)
- Three.js (CDN importmap)
- CSS custom properties for theming
- PageController class for navigation
